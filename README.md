# Fare Update Project (Frontend)

This is the frontend for the Fare Update Project. It's a React application built with Vite, TypeScript, shadcn-ui, and Tailwind CSS.

## Getting Started

1.  **Clone the repository:**
    ```sh
    git clone https://github.com/Sishir-Stha/fare-manager-pro.git
    cd fare-manager-pro/frontend
    ```
2.  **Install dependencies:**
    ```sh
    npm install
    ```
3.  **Run the development server:**
    ```sh
    npm run dev
    ```

## Technologies Used

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
